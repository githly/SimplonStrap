 Le Reset User Agent 

        
        
 Le Reset User Agent permet de réinitialiser les paramètres d'affichage par défaut du navigateur que l'on ne souhaite pas conserver dans notre code 

        

          

            
 Le Css 

            
              /* -----------------------------------------------Reset User Agent */

              * {
                margin: 0; /* Reintialise all margin*/
                padding: 0; /* Reintialise all padding*/
                list-style-type: none; /* Delete Bullet on list */
                text-decoration: none; /* Delete Underline on Link */
                box-sizing: border-box; /* Padding and Border include in box size */
              }

              img {
                max-width: 100%;
              }
